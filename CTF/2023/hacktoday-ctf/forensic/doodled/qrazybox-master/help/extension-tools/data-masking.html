Data Masking


Data Masking is a simple tool to mask/unmask (performing XOR operation) modules of your current work according to the specified mask pattern. It can be used when you want to analyze QR code that not masked normally or when the QR code is already unmasked.

Selecting this tool will open new dialog and you should select mask pattern from 0 to 7 that you want mask/unmask on your current work.
