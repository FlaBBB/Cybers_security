Extension Tools


Extension Tools are collection of advanced individual tool that helps you analyze and recover your QR code according to the problems that being encountered in your work.
You can access these tools by clicking Tools in main menu.


QRazyBox provides following extension tools :

  • Extract QR Information

  • Reed-Solomon Decoder

  • Brute-force Format Info Pattern

  • Data Unmasking

  • Padding Bits Recovery
